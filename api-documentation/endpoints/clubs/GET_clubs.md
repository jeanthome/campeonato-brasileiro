
# Recurso: Clubes

## URI 
    GET /clubs
***

## Descrição
Retorna todos os clubes participantes da temporada atual do campeonato.
***

## Parâmetros
Nenhum.
***

## Formato de retorno
Um objeto JSON com uma lista de clubes encontrados.
***

    https://domain.com/clubs
